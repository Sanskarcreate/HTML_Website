Things you need to know?

The main thing you need to know the books we have in the library is that the books have a unique number on the back of each book which allows us to identify the books and if they belong to us. So before purchasing or returning a book

PLEASE MAKE SURE THAT THE NUMBER IS THERE OR ELSE WE WILL NOT BE ABLE TO PROCESS YOUR REQUEST!!!

Also if this is the case with a purchased book then we would not be able to give a REFUND as we would not know if that book is ours since it does not have a number at the back.
